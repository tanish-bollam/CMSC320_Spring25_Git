Q1: Why might we consider using Git?
Answer: We would consider to use Git because it helps with version control. In specific, we will use git to track changes in code, collaborate efficiently, and manage different versions of a project.

Q3: What is the difference between Merging and Rebasing?
Answer: The difference between merging and rebasing is that merging creates a new commit that combines the changes while preserving the history, while rebasing rewrites the commit history applying changes from one branch on top of another.
